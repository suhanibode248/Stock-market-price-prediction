## Abstract  
Stock market prediction is an important task for investors and financial analysts. This project focuses on predicting stock prices using machine learning techniques. The goal is to analyze past stock data and forecast future prices to support better decision-making.  

The system uses historical data such as opening price, closing price, volume, and trends. The data is first cleaned and prepared, and then machine learning models like Linear Regression and Random Forest are applied.  

The results show that the model can predict stock price trends with reasonable accuracy. This project demonstrates how data-driven approaches can help in understanding stock market behavior and making informed investment decisions.  

---

## Introduction  
The stock market is highly dynamic and influenced by many factors such as economic conditions, company performance, and market trends. Predicting stock prices is challenging but very useful for investors.  

The main objective of this project is to build a system that can predict future stock prices based on historical data. By analyzing past trends, the system can help users make better financial decisions.  

This project is important because it provides a simple way to understand market patterns and supports smarter investment strategies.  

---

## Literature Review  
Many researchers have worked on stock price prediction using statistical and machine learning methods. Traditional approaches include time series analysis, while modern methods use machine learning algorithms like Linear Regression, Random Forest, and LSTM (deep learning).  

Some models focus only on historical price data, while others include external factors like news and market sentiment.  

In this project, basic machine learning models are used to understand trends and provide reliable predictions.  

---

## Methodology  
The system uses historical stock data as input. First, the data is cleaned by removing missing values and errors. Then, it is normalized for better model performance.  

Important features like open price, close price, and volume are selected. The dataset is divided into training and testing sets.  

Machine learning models like Linear Regression and Random Forest are trained on the data. The models are evaluated using performance metrics, and the best model is selected for predicting future stock prices.
